## Installasjon

### Backend

```bash
cd Full-stack-prosjekt\backend
# With Maven  

# With Docker-compose (?)
docker-compose -f docker-compose build
docker-compose -f docker-compose up
```

### Frontend

```bash
cd Full-stack-prosjekt\frontend
# Run the app
npm run serve
```
